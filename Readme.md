# Поиск пароля к архиву

## Описание задачи
Задача подобрать пароль к архиву secured.zip, пароли берем из файла passwords.txt, где каждая строка это пароль.

   <groupId>org.apache.logging.log4j</groupId>

 </dependency>
 <dependency>

 </dependency>
